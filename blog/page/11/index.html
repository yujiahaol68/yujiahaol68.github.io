CSS学习笔记02 nav上下居中

November 1, 2016 Tecker.Yu

1 minute read

.nav-desktop{
	background-color: #e83a37;
	font-size: 18px;
	padding-top: 0px;
	padding-bottom: 0px;
	height: 60px;
}

.menu-text {
	line-height: 60px;
	padding: 0px;
}

只需要调整line-height与nav导航栏的高度一致，将padding设置为0，就能够轻松实现上下居中
