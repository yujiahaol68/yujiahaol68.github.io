Sass之CSS模块化

December 3, 2016 Tecker.Yu

1 minute read

  • Twitter

  • Google+

  • Facebook

  • Reddit

  • LinkedIn

  • StumbleUpon

  • Pinterest

  • Email

一般在项目目录中我们会建一个sass文件夹，内容一般为partials文件夹还有一个app.scss文件，partials文件夹存放我们的CSS模块，app.scss中进行import

@import 'partials/temp;'

partials文件夹中的模块scss文件最前面必须加上下划线 _

使得sass在监听并编译成css的时候能够将模块编译到一起而不是生成多个css文件

CSS模块常用目录结构

第一种方式

  1. base 基本样式
    • variable 变量
    • typography
    • base 例如:wrapper居中CSS
  2. layout 布局
    • nav
    • hero
    • footer
    • layout
  3. component 组件
    • button
    • schedule
